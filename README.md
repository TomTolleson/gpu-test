## Project Description
A project to benchmark GPU performance using Hugging Face models.

## Prerequisites
- Python 3.x
- PyTorch
- Transformers
- CUDA-capable GPU

## Usage
```bash
# Install dependencies      
pip install -r requirements.txt

# Install the package
python -m build

# Run the benchmark script
python tests/gpu_benchmark.py
```
